🎮 项目说明：自研游戏架构与反作弊机制

🧠 项目愿景

本项目旨在构建一个完全自主、可扩展、社区驱动的游戏架构。无论是否商业化，开发者都能掌控分发、账号体系、反作弊机制与玩家自治系统。

🧱 架构概览

  • 启动流程：verify() → Awake() → 多线程初始化
  • 反作弊机制：DLL校验 + 行为评分 + 主动崩溃
  • 账号体系：支持微软账号 / Steam账号登录（PlayFab集成）
  • 分发方式：静态网站 + CDN 国内外同步
  • 赞助系统：免费本体 + 标牌激励（爱发电）

🔐 反作弊机制

采用三层防护：

  1. 启动校验：DLL完整性 + 签名验证
  2. 事件监听：异常行为触发主动崩溃
  3. 社区治理：举报系统 + 回放审查 + 投票踢人

🧩 技术栈与轮子复用

  • 网络通信：LiteNetLib / Mirror
  • 账号系统：PlayFab
  • 数据处理：Newtonsoft.Json / MessagePack
  • UI框架：Unity UI Toolkit
  • 日志系统：Serilog

📜 授权策略

若未获原作者授权：游戏本体免费，赞助仅提供标牌。

若获得授权：游戏可转为买断制，赞助系统保留。

🎓 毕业设计定位

本项目作为计算机专业毕业设计，具备完整的架构设计、模块实现、反作弊机制与社区治理系统，技术深度与工程完整度兼备。
